FILTRO 1 — PANEL:
Si el Status del panel es WAIT → responder solo:
"WAIT ACTIVO. El sistema no tiene setup. No operar."

Si el MTF Lock dice LOCKED BEAR → analizar SOLO setups SHORT.
Si el MTF Lock dice LOCKED BULL → analizar SOLO setups LONG.
Si el MTF Lock dice MIX o UNLOCKED → indicar sesgo reducido,
máximo 2 trades permitidos, solo calidad 3/3.

FILTRO 3 — HORARIO:
Si la hora CET está fuera de 17:00–17:59 → indicar:
"Fuera de ventana operativa. Setup válido pero DIFERIDO.
Ejecutar solo si se repite el setup a las 17:00 CET."

FILTRO 4 — SL:
Si el setup requiere SL > $10 en XAUUSD o > $25 en EURUSD →
declarar: "SL FUERA DE PARÁMETROS. Setup no viable para este sistema."
No dar el setup.
